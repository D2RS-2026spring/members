##2. 如果电脑坏了，你能恢复多少数据？ 
代码和文档：由于代码和 Markdown 笔记都通过 Git 推送到 GitHub，可 100% 恢复。  
分析结果和图表：保存在本地，但每周同步到移动硬盘，可恢复最近一周内的结果。  
原始数据：部分原始数据来源于公开数据库，可重新下载；但一些私有数据如果移动硬盘也未备份，可能会丢失，恢复率约 80%。
