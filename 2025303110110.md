##数据存储与备份方案
1. 数据目前存放位置
实验原始数据、分析代码：主要存放在电脑本地D盘的WORK文件夹。
文献资料、实验方案、开题报告：存放在电脑本地文档库，同步至百度网盘便于跨设备查阅。
核心分析代码、Git协作作业文件：存放在GitHub仓库（含本次members作业仓库）。
实验现场照片、显微镜图像：存放在手机本地，同步至iCloud云相册备用。

2. 电脑损坏时的数据恢复能力
实验代码与 Git 作业文件：可从 GitHub 完整恢复最新版本，无丢失风险。
文献、报告类文档：可从百度网盘恢复全部已同步内容，包含历史修改版本。
实验原始数据：可从外接移动硬盘恢复最近一次备份的核心数据，仅可能丢失未及时备份的当日临时实验记录。
图像类资料：可从iCloud云相册完整恢复，不受电脑损坏影响。
