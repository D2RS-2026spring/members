1.你的数据目前存放在哪里？
(1)主要数据存放在：电脑本地硬盘（如 D:\work、C:\Users\34593\Documents）；
(2)部分代码和文档同步到：GitHub 仓库；
(3)少量照片和文件：手机相册 + 百度云盘。

2.如果电脑坏了，你能恢复多少数据？
(1)已同步到 GitHub / 云端的代码和文档：100% 可恢复；
(2)本地未备份的最新修改、未同步的文件：可能丢失，无法恢复；
(3)照片和聊天记录：已经开启了云同步，大部分可恢复。

3.制定一个 3-2-1 备份方案：
(1)主工作目录：D:\work（或 ~/work），存放当前正在编辑的所有项目和文件；
(2)本地备份：外接 1TB 移动硬盘，每周自动同步一次 D:\work 目录；
(3)云端备份：
代码和文档：GitHub + 坚果云（实时同步）；
照片和重要文件：iCloud / OneDrive。
