## 2 存储备份策略

### 2.1 你的数据目前存放在哪里

- 课程核心文件（课件、代码）存放在本地电脑 D 盘 course 目录中

- 个人资料分散在本地电脑 C 盘与 D 盘中

### 2.2 如果电脑坏了，你能恢复多少数据？

- 电脑若仅为系统故障，可尝试恢复本地硬盘中的文件，但恢复后整理成本较高
- 硬盘若物理损坏或电脑丢失，在无备份的情况下本地数据将彻底丢失
- 只有已提交到 GitHub 等平台的代码和作业才可以恢复

### 2.3 制定一个 3-2-1 备份方案：

- 主工作目录：本地电脑D盘course目录

- 本地备份：使用移动硬盘定期同步数据

- 云端备份：核心代码和作业同步到 GitHub
